## VS Code Docker Extension Nedir?

<br>

Docker uzantısı, kod düzenleyicinizden ayrılmadan kapsayıcılı uygulamalar yaratmayı, yönetmeyi ve dağıtmayı kolaylaştırır. Basitçe söylemek gerekirse Docker'ı daha iyi yönetmenize yardımcı olur.

<br>
<br>

## VS Code YAML Extension Nedir?

<br>

YAML uzantısı, doğrulama, hata algılama, belge taslağı yaratma, otomatik tamamlama, fareyle üzerine gelme desteği ve ayrıca Kubernetes YAML dosyaları için söz dizimi denetimi sağlar.
